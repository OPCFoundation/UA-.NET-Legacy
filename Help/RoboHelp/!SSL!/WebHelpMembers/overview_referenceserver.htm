Reference Server Overview

The Reference Server is a purpose built server used during the development and testing of the Unified Architecture UA CTT (Compliance Test Tool). The address-space of this server has been designed to match the design of the UA CTT configuration.

Intended for providing a simple reference implementation when learning the UA CTT.

This server is based on the Quickstarts framework and therefore inherits the same capabilities, features, and functionality found in the Data Access Server, Historical Data Server, Historical Events Server, and Alarms & Conditions Server.

This server may be configured using the same guidelines as described for any of the applications listed above. You may refer to the Data Access Server Configuration as an example.
